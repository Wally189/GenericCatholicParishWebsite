# Generic Catholic Parish Website

An anonymised Roman Catholic parish website template built with plain HTML, CSS, and JavaScript, with editable content managed through Decap CMS.

## What this template includes

- Homepage with hero slideshow, Mass summary, news, mission, parish life, and contact sections
- Shared header and footer
- Responsive navigation
- Editable JSON content under `content`
- Placeholder newsletters, documents, and image assets under `uploads`
- Decap CMS admin area at `admin/index.html`
- Help guides for non-technical editors

## Template structure

- Public layouts live in the root HTML files such as `index.html` and `mass-services.html`.
- Shared editable content lives in `content/settings`.
- Page-specific editable text lives in `content/pages`.
- News, newsletters, and downloadable documents live in:
  - `content/news/items.json`
  - `content/newsletters/items.json`
  - `content/documents/items.json`
- Uploaded images and PDFs live in `uploads`.

## Before using this template

Replace the placeholder parish name, contact details, Mass times, documents, and images with the real parish content before launch.
